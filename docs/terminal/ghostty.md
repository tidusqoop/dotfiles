# Ghostty

GPU 가속 터미널 에뮬레이터. Zig 기반, 네이티브 macOS 지원.

## 설정 파일

- 경로: `~/.config/ghostty/config`
- chezmoi: `dot_config/ghostty/config`

## 단축키 (macOS 기본)

### 탭

| 단축키 | 동작 |
| --- | --- |
| `Cmd + T` | 새 탭 |
| `Cmd + W` | 탭 닫기 |
| `Cmd + Shift + [` | 이전 탭 |
| `Cmd + Shift + ]` | 다음 탭 |
| `Cmd + 1~9` | n번째 탭으로 이동 |

### 분할 (Split)

| 단축키 | 동작 |
| --- | --- |
| `Cmd + D` | 수평 분할 |
| `Cmd + Shift + D` | 수직 분할 |
| `Cmd + W` | 현재 분할 닫기 |
| `Cmd + Option + 방향키` | 분할 포커스 이동 |
| `Cmd + Shift + Enter` | 분할 최대화 토글 |

### 폰트 / 화면

| 단축키 | 동작 |
| --- | --- |
| `Cmd + +` | 폰트 크기 증가 |
| `Cmd + -` | 폰트 크기 감소 |
| `Cmd + 0` | 폰트 크기 리셋 |
| `Cmd + Ctrl + F` | 전체 화면 토글 |

### 텍스트 / 클립보드

| 단축키 | 동작 |
| --- | --- |
| `Cmd + C` | 복사 |
| `Cmd + V` | 붙여넣기 |
| `Cmd + Shift + V` | 서식 없이 붙여넣기 |
| `Cmd + F` | 검색 |
| `Cmd + K` | 화면 지우기 (clear) |

### 기타

| 단축키 | 동작 |
| --- | --- |
| `Cmd + N` | 새 창 |
| `Cmd + ,` | 설정 열기 |
| `Cmd + Shift + ,` | 설정 리로드 |
| `Cmd + Q` | 종료 |
