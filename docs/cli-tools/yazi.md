# yazi 사용법

> Rust 기반 터미널 파일 매니저. 비동기 I/O, Vi 키바인딩, 파일 미리보기를 지원하는 모던 TUI 파일 탐색기.

## 실행

```bash
# 실행
yazi

# 특정 디렉토리에서 실행
yazi ~/projects

# yazi 종료 시 마지막 디렉토리로 이동 (셸 래퍼)
function y() {
  local tmp="$(mktemp -t "yazi-cwd.XXXXXX")" cwd
  yazi "$@" --cwd-file="$tmp"
  if cwd="$(command cat -- "$tmp")" && [ -n "$cwd" ] && [ "$cwd" != "$PWD" ]; then
    builtin cd -- "$cwd"
  fi
  rm -f -- "$tmp"
}
```

## 핵심 키바인딩

### 이동

| 키 | 설명 |
|----|------|
| `h` / `l` | 상위 / 하위 디렉토리 이동 |
| `j` / `k` | 아래 / 위 이동 |
| `J` / `K` | 5줄씩 아래 / 위 이동 |
| `gg` / `G` | 처음 / 끝으로 이동 |
| `~` | 홈 디렉토리로 이동 |
| `-` | 이전 디렉토리로 돌아가기 |

### 파일 작업

| 키 | 설명 |
|----|------|
| `<Enter>` | 파일 열기 |
| `o` | 기본 프로그램으로 열기 |
| `e` | 에디터로 열기 |
| `y` | 복사 (yank) |
| `x` | 잘라내기 (cut) |
| `p` | 붙여넣기 (paste) |
| `d` | 휴지통으로 이동 |
| `D` | 영구 삭제 |
| `a` | 파일/디렉토리 생성 |
| `r` | 이름 변경 |

### 선택

| 키 | 설명 |
|----|------|
| `<Space>` | 현재 항목 선택/해제 |
| `v` | 비주얼 모드 (범위 선택) |
| `V` | 비주얼 모드 해제 |
| `<C-a>` | 전체 선택 |
| `<C-r>` | 선택 반전 |

### 탭

| 키 | 설명 |
|----|------|
| `t` | 새 탭 |
| `1`-`9` | n번 탭으로 이동 |
| `[` / `]` | 이전 / 다음 탭 |
| `<C-c>` | 탭 닫기 |

### 검색/필터

| 키 | 설명 |
|----|------|
| `/` | 검색 (fd 연동) |
| `f` | 필터 (실시간 필터링) |
| `s` | 셸 명령어 실행 |
| `S` | 블로킹 셸 명령어 실행 |
| `z` | zoxide로 디렉토리 점프 |
| `Z` | fzf로 디렉토리 점프 |

### 보기

| 키 | 설명 |
|----|------|
| `.` / `<C-h>` | 숨김 파일 토글 |
| `,` | 정렬 방식 변경 |
| `w` | 태스크 매니저 열기 |
| `<C-z>` | yazi를 백그라운드로 (fg로 복귀) |

## 미리보기 지원

yazi는 오른쪽 패널에서 파일 미리보기를 제공한다:

| 파일 타입 | 미리보기 | 필요 도구 |
|-----------|----------|-----------|
| 텍스트/코드 | 구문 강조 | bat (설치됨) |
| 이미지 | 썸네일 | 내장 |
| 동영상 | 썸네일 | ffmpeg |
| PDF | 첫 페이지 | poppler |
| 압축 파일 | 파일 목록 | sevenzip |
| 디렉토리 | 파일 목록 | 내장 |

## 설정 파일

```
~/.config/yazi/
├── yazi.toml      # 일반 설정 (정렬, 미리보기 등)
├── keymap.toml    # 키바인딩 커스터마이징
└── theme.toml     # 테마 설정
```

## 실전 활용

### 파일 복사/이동 워크플로우

```
1. 소스 디렉토리에서 파일 선택 (<Space>로 다중 선택)
2. y (복사) 또는 x (잘라내기)
3. 목적지 디렉토리로 이동
4. p (붙여넣기)
```

### 빠른 디렉토리 탐색

```
1. z 입력 → zoxide 키워드로 점프
2. / 입력 → fd 기반 파일 검색
3. f 입력 → 현재 디렉토리 실시간 필터링
```

### 셸 명령어 실행

```
1. s 입력 → 명령어 입력 후 실행 (논블로킹)
2. S 입력 → 명령어 입력 후 실행 (블로킹, 결과 확인 가능)
```

## 사용 팁

- **`y` 함수를 `.zshrc`에 추가**하면 yazi 종료 후 마지막 탐색 위치로 셸이 이동한다
- `z` 키로 zoxide 연동이 되므로 yazi 안에서도 키워드 기반 점프가 가능하다
- 대량 파일 작업 (복사, 이동, 삭제)은 `<Space>`로 다중 선택 후 일괄 처리한다
- `w` 키로 진행 중인 파일 작업(복사 등)의 진행률을 확인할 수 있다
